DOGL WEBSITE  (recommended starting point)
==========================================
This folder IS your website — a faithful rebuild of your current site,
off Wix and fully yours. Start here.

To put it online, follow the setup guide:  https://dogl-guide.pages.dev

Quick start: open this folder in Claude Code and say:
  "Help me publish this website to GitHub Pages, one step at a time."

Later, you can add anything you like (searchable data tables, charts,
new pages) just by asking Claude Code — on your own terms.
